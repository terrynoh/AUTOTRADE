# AUTOTRADE Master-Slave 아키텍처 전환 계획

**문서 버전**: v1.0  
**작성일**: 2026-04-11  
**상태**: 계획 단계 (DRY_RUN → live 검증 완료 후 착수)

---

## 1. 개요

### 1.1 목적
- 다중 사용자 지원 (가족 계좌 등)
- 계좌 키 보안 강화 (클라우드에 KIS 시크릿 미보관)
- 장애 격리 (A 장애 ≠ B 영향)
- 법률적 방어 논리 확보 (정보 제공 vs 대리 운용)

### 1.2 핵심 원칙
```
클라우드 = 두뇌 (종목 선정)
로컬 = 손발 (매매 실행)
```

|---|---|---|
| `master/screener.py` | 종목 선정 | 기존 Screener 재사용 |
| `master/api.py` | REST API | FastAPI |
| `master/dashboard.py` | 통합 대시보드 | 모든 Slave 상태 표시 |

```

GET  /api/slaves           → 연결된 Slave 목록
```

## 5. 코드 변경 예시

### 5.1 Master: /api/targets
```python
# master/api.py
from fastapi import FastAPI, Depends, HTTPException
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials

app = FastAPI(title="AUTOTRADE Master")
security = HTTPBearer()

# 인메모리 상태 (또는 Redis)
selected_targets: dict = {
    "codes": [],
    "ts": None,
    "selected_by": "screener"  # or "manual"
}

def verify_token(creds: HTTPAuthorizationCredentials = Depends(security)):
    valid_tokens = os.getenv("SLAVE_TOKENS", "").split(",")
    if creds.credentials not in valid_tokens:
        raise HTTPException(401, "Invalid token")
    return creds.credentials

@app.get("/api/targets")
async def get_targets(token: str = Depends(verify_token)):
    return selected_targets

@app.post("/api/targets")
async def set_targets(codes: list[str], token: str = Depends(verify_token)):
    """수동 종목 설정 (대시보드/텔레그램)"""
    selected_targets["codes"] = codes
    selected_targets["ts"] = now_kst().isoformat()
    selected_targets["selected_by"] = "manual"
    return {"ok": True, "count": len(codes)}
```

### 5.2 Slave: sync.py
```python
# slave/sync.py
import httpx
from loguru import logger

class MasterSync:
    def __init__(self, master_url: str, token: str, slave_id: str):
        self.master_url = master_url.rstrip("/")
        self.token = token
        self.slave_id = slave_id
        self._client = httpx.AsyncClient(
            headers={"Authorization": f"Bearer {token}"},
            timeout=10.0
        )
        self._last_ts: str = ""
    
    async def fetch_targets(self) -> list[str]:
        """Master에서 종목 목록 가져오기"""
        try:
            resp = await self._client.get(f"{self.master_url}/api/targets")
            resp.raise_for_status()
            data = resp.json()
            
            # 변경 감지
            if data["ts"] != self._last_ts:
                self._last_ts = data["ts"]
                logger.info(f"[Sync] 종목 갱신: {data['codes']}")
            
            return data["codes"]
        except Exception as e:
            logger.error(f"[Sync] Master 통신 실패: {e}")
            return []  # 실패 시 빈 리스트 (기존 종목 유지는 호출자 판단)
    
    async def report_execution(self, code: str, side: str, price: int, qty: int):
        """체결 내역 Master에 전송"""
        try:
            await self._client.post(
                f"{self.master_url}/api/executions",
                json={
                    "slave_id": self.slave_id,
                    "code": code,
                    "side": side,
                    "price": price,
                    "qty": qty,
                    "ts": now_kst().isoformat()
                }
            )
        except Exception as e:
            logger.warning(f"[Sync] 체결 보고 실패 (무시): {e}")
```

### 5.3 Slave: main.py 수정
```python
# slave/main.py (수정 부분만)

class AutoTrader:
    def __init__(self, ...):
        # 기존 코드 ...
        
        # 신규: Master 동기화
        self._sync = MasterSync(
            master_url=settings.master_url,
            token=settings.master_token,
            slave_id=settings.slave_id
        )
        self._last_targets: list[str] = []
    
    async def _poll_master_targets(self):
        """30초마다 Master에서 종목 폴링"""
        while self._running:
            targets = await self._sync.fetch_targets()
            
            if targets and targets != self._last_targets:
                self._last_targets = targets
                self.set_manual_codes(targets)
                await self._on_screening()  # 스크리닝 실행
            
            await asyncio.sleep(30)
    
    async def _on_buy_filled(self, code: str, price: int, qty: int):
        # 기존 체결 처리 ...
        
        # Master에 보고
        await self._sync.report_execution(code, "buy", price, qty)
```

## 7. 리스크 및 대응

| 리스크 | 확률 | 대응 |
|---|---|---|
| Master 장애 | 낮음 | Slave는 마지막 종목으로 계속 운영 |
| 네트워크 단절 | 중간 | 폴링 실패 시 기존 상태 유지 |

| 토큰 유출 | 낮음 | Slave별 개별 토큰 + 즉시 폐기 |
