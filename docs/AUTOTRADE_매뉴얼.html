2. 대시보드 접속 방법

로컬 접속 (같은 PC)

브라우저에서 http://localhost:8503 접속 → 자동으로 관리자 권한 부여

원격 접속 (텔레그램 URL)

매일 09:00에 텔레그램으로 전송되는 Cloudflare Tunnel URL로 접속합니다.

AUTOTRADE 대시보드 시작
시각: 2026-04-07 09:00 KST
모드: dry_run

관리자 접속:
https://xxx-yyy.trycloudflare.com?token=d952...

읽기 전용:
https://xxx-yyy.trycloudflare.com
관리자 vs 읽기 전용
- 관리자 URL (?token= 포함): 종목 설정, 스크리닝, 연결 등 모든 기능 사용 가능
- 읽기 전용 URL: 모니터링만 가능 (버튼 숨김)

4. 대시보드 수동 종목 설정 (Step by Step)

1
대시보드 접속
텔레그램에서 받은 관리자 URL 클릭, 또는 http://localhost:8503 접속
2
KIS API 연결 확인
헤더에 ● Connected가 표시되어야 합니다.
Disconnected이면 API 연결 버튼 클릭
3
종목 검색 또는 직접 입력
방법 A: 상단 검색창에 삼성 입력 → 드롭다운에서 삼성SDI(006400) 클릭 → 파란 태그 추가
방법 B: 하단 직접 입력란에 006400, 에코프로비엠, 020150 입력
4
"종목 설정" 버튼 클릭
KIS API로 종목 유효성 검증 → 하단에 설정된 종목: 삼성SDI(006400), ... 표시
5
"수동 스크리닝" 버튼 클릭
설정된 종목에 대해 5단계 필터 실행:
① 상승률 > 0% → ② 거래대금 ≥ 500억 → ③ 상한가 미도달 → ④ 프로그램순매수비중 ≥ 5% → ⑤ 상승률 최고 종목
통과 종목이 타겟 종목 모니터에 표시됩니다.
6
자동 매매 진행
이후 자동으로 신고가 감시 → 매수 → 청산까지 진행됩니다.
텔레그램으로 매수/매도 알림이 전송됩니다.
주의사항
- 종목 설정은 09:50 전에 완료하세요. 09:50에 자동 스크리닝이 실행됩니다.
- 종목 설정 없이 09:50을 지나면 자동 스크리닝(거래량 순위 기반)으로 폴백됩니다.
- 종목은 최대 10개까지 입력 가능합니다 (필터 후 최대 5종목 선정).

9. FAQ / 주의사항

Q: 종목은 몇 개까지 입력할 수 있나요?

최대 10개까지 입력 가능합니다. 스크리닝 후 최대 5종목이 후보 풀로 선정됩니다.

Q: 종목명으로 검색이 안 되는 종목이 있어요

종목 마스터(~2,800종목)에 없는 종목은 6자리 코드로 직접 입력하세요. ETF/ETN은 코드 입력을 권장합니다.

Q: 대시보드 URL이 매일 바뀌나요?

네. Cloudflare Quick Tunnel은 매번 새 URL을 발급합니다. 매일 09:00에 텔레그램으로 새 URL이 전송됩니다.

Q: DRY_RUN 모드에서는 실제 돈이 빠지나요?

아니요. DRY_RUN은 완전한 가상 매매입니다. 실제 주문이 전송되지 않으며 가상 예수금(5천만원)으로 시뮬레이션합니다.

Q: 텔레그램 명령은 누가 사용할 수 있나요?

.env의 TELEGRAM_CHAT_ID에 등록된 사용자만 명령을 실행할 수 있습니다. 미등록 사용자의 명령은 무시됩니다.

Q: 매매 중 네트워크가 끊기면?

WebSocket 끊김 감지 시 자동으로 미체결 매수 주문을 취소합니다. 이미 보유 중인 포지션은 텔레그램으로 수동 청산 안내가 발송됩니다.

이 시스템은 투자 자문을 제공하지 않습니다. 모든 매매 결과에 대한 책임은 사용자에게 있습니다. 반드시 DRY_RUN → PAPER(모의투자) 순서로 충분히 검증한 후 실매매로 전환하세요.
