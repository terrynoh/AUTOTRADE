# M-23 — 프로그램 순매수 비중 실시간 업데이트 진단

> **목적**: R-13 구현 전 기존 로직 보호하면서 비중 실시간 업데이트를 "얹을" 수 있는 지점 식별
> **작성**: 2026-04-16
> **원칙**: 기존 로직 유지 + 추가 개념 (신규 로직 아님)

---

## 배경

### 현재 문제
- 프로그램 순매수 비중이 09:50 스크리닝 시점에 1회만 조회됨
- 이후 비중 변동 (예: 29% → 9%) 반영 안 됨
- SKC 케이스: Double 기준 손절(-3.0%)로 매수 기회 놓침

### R-13 목표
- 비중을 실시간으로 업데이트 (1초 단위)
- 비중 변경 시 매수가/손절가 재계산
- 1차 체결 후 비중 변경 시 2차 취소 → 재발주

### 핵심 원칙 (수석님 확정)
1. 현재 프로그램 **잘 작동 중** — 기존 로직 보호 필수
2. 기존 로직 **해치면 안 됨** — 오류 발생 금지
3. **업데이트 개념** — 신규 로직 아님
4. 기존 로직에 **추가**되어 더 잘 작동하는지가 목표
